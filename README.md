## üìñ Introduction

The AI-Powered Interview Assistant is a comprehensive solution built to streamline and enhance the technical screening process. It addresses the time-consuming nature of initial interviews by providing a fully automated, yet deeply insightful, platform. Candidates can take a timed, AI-driven interview at their convenience, while recruiters and hiring managers receive a detailed report, including a technical score, a performance summary, and advanced vocal analysis. This allows teams to focus their efforts on the most promising candidates.

---

## ‚ú® Key Features

-   **Role-Based Landing Page:** Separate, tailored entry points for Candidates and Interviewers.
-   **Automated Resume Parsing:** Automatically extracts key candidate details (Name, Email, Phone) from uploaded PDF resumes.
-   **Dynamic AI Question Generation:** Generates relevant interview questions in real-time based on selected difficulty (Easy, Medium, Hard).
-   **Hybrid Input System:** Intelligently offers voice input for conceptual questions and text input for problem-solving, providing a flexible user experience.
-   **Timed Interview Flow:** Each question is timed to simulate a real-world screening environment, with answers submitted automatically when time expires.
-   **Advanced Vocal Analysis:** Leverages the AssemblyAI API to analyze a candidate's speech for sentiment, confidence, and the use of filler words, offering deeper communication insights.
-   **AI-Powered Scoring & Summary:** Upon completion, a powerful AI model provides a final score and a concise, insightful summary of the candidate's performance.
-   **Interviewer Dashboard:** A clean, comprehensive dashboard for interviewers to view all completed candidate interviews.
-   **Search & Sort Functionality:** Interviewers can instantly search for candidates by name and sort the list by final score.
-   **Detailed Interview Review:** A modal view allows for a deep dive into a candidate's full interview transcript, including every question, answer, and the associated vocal analysis.
-   **Local State Persistence:** All session and candidate data is saved locally using Redux Persist. Users can refresh the browser or return later and resume an in-progress interview.

## üöÄ Getting Started

Follow these instructions to set up and run the project on your local machine for development and testing purposes.

### Prerequisites

-   [Node.js](https://nodejs.org/en/) (v18.x or later recommended)
-   [npm](https://www.npmjs.com/)

### Local Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/ai-powered-interview-assistant.git
    cd ai-powered-interview-assistant
    ```

2.  **Install dependencies:**
    This project is a monorepo with separate dependencies for the root, client, and server.

    ```bash
    # Install root dependencies
    npm install

    # Install server dependencies
    npm install --prefix server

    # Install client dependencies
    npm install --prefix client
    ```

3.  **Set up environment variables:**
    Create a `.env` file in the `server/` directory and add your API keys.

    ```
    # server/.env

    # Key for parsing resumes (e.g., APILayer)
    APILAYER_API_KEY="your_apilayer_key_here"

    # Key for AI question generation and evaluation (e.g., OpenRouter)
    OPENROUTER_API_KEY="your_openrouter_key_here"

    # Key for vocal analysis
    ASSEMBLYAI_API_KEY="your_assemblyai_key_here"
    ```

4.  **Run the application:**
    Start both the client and server concurrently from the root directory.

    ```bash
    npm start
    ```

    The client will be available at `http://localhost:5173` and the server at `http://localhost:5000`.
